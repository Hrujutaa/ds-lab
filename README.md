Data Structures Laboratory (2nd Semester)

A comprehensive implementation of fundamental Data Structures and Algorithms using C. This repository demonstrates structured programming, memory management, and algorithmic problem-solving techniques essential for building efficient software systems.

--Overview

This laboratory focuses on:

* Implementation of linear and non-linear data structures
* Algorithm design and analysis
* Time and space complexity evaluation
* Dynamic memory management using pointers
* Practical problem-solving techniques

All programs are written in C and compiled using a standard C compiler.

--Topics Covered

 1. Arrays

* Static arrays
* Dynamic arrays
* Traversal, insertion, deletion
* Searching algorithms:

  * Linear Search
  * Binary Search
* Sorting algorithms:

  * Bubble Sort
  * Selection Sort
  * Insertion Sort

Key Concepts:

* Contiguous memory allocation
* Index-based access (O(1))
* Time complexity comparison

 2. Linked Lists

* Singly Linked List
* Doubly Linked List
* Circular Linked List
* Operations:

  * Insertion (Beginning, End, Position)
  * Deletion
  * Traversal
  * Searching

 Core Understanding:

* Dynamic memory allocation (`malloc`, `free`)
* Pointer manipulation
* Memory efficiency vs arrays

 3. Stack (LIFO Structure)

* Array-based implementation
* Linked list implementation
* Operations:

  * Push
  * Pop
  * Peek

 Applications:

* Parenthesis validation
* Expression evaluation
* Infix → Postfix conversion

 4. Queue (FIFO Structure)

* Linear Queue
* Circular Queue
* Basic Dequeue concept
* Operations:

  * Enqueue
  * Dequeue

 Applications:

* Scheduling systems
* Buffer handling

 5. Trees

* Binary Tree
* Binary Search Tree (BST)
* Tree Traversals:

  * Inorder
  * Preorder
  * Postorder

 Concepts:

* Recursive implementation
* Hierarchical data modeling
* BST search efficiency (O(log n) average case)

 6. Graphs (Introduction)

* Graph representation:

  * Adjacency Matrix
  * Adjacency List
* Graph Traversal:

  * Breadth First Search (BFS)
  * Depth First Search (DFS)

 7. Recursion

* Recursive functions
* Stack memory behavior
* Recursive tree traversal
* Time complexity impact


  Tech Stack

* Language: C
* Compiler: GCC / Turbo C / Any Standard C Compiler
* IDE: VS Code / CodeBlocks / Dev C++
